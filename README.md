- 매출액이 2,197억원으로 사상 최대치를 기록하며 해상풍력관련 매출비중은 대폭 높아짐
- 일회성 야드 보강비용 등으로 영업이익률은 전년동기 대비 하락하였으나, 일회성 비용이 적게 발생해 추정치를 상회함
- 과세부과처분에 대한 경정청구로 76억원의 일회성 법인세 환급을 받아 당기순이익 예상치 및 컨센서스를 100% 이상 상회
- 연간 유효법인세율은 7.53%로 전망
