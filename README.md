다양한 제품 라인업을 보유하고 있으며, 2개 이상의 멀티 시술 방식이 최근 피부미용기기 시술 트렌드이기 때문에 수요자 대응이 용이하다. 이는 국내경쟁사 대비 프리미엄을 받을 수 있는 요인이라는 것으로 나타났다. 유안타증권은 원텍에 대해 Buy를 유지하고 목표 주가를 8,000원으로 설정했다.
