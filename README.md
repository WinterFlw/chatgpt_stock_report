## 한화오션
한화오션은 3분기에 매출 1조 9,169억원과 영업이익 741억원을 기록하여 분기 흑자 전환을 달성했다. 이는 3년 만에 분기 흑자 전환으로, 일회성 이익과 요인을 제외한 경상 이익은 61억원으로 나타났다. 분기 흑자 전환에 따라 SK증권은 한화오션을 매수 추천하며 목표 가격을 30,000원으로 설정했다.
## 삼성바이오로직스
