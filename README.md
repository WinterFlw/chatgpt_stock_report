- 이마트의 신세계건설 PF 충당금 설정은 본업의 확장에도 영향을 미칠 수 있으며, 연결부채 증가로 인한 신용등급 하락과 금융비용 증가, 할인점 오프라인 출점 전략이 지연됨으로써 성장성 확보를 위한 투자가 보류될 가능성도 있다. 자구책을 마련할 경우에는 최악의 상황을 피할 수도 있지만, 위험도 높은 상황일 수 있으므로 주의가 필요하다.

- 4분기 2023 영업손실은 2,420억원으로 예상되며, 전분기 대비 크게 개선될 것으로 예상됨.

- AI와 서버 중심의 수요 집중 현상이 계속되며, 1분기 2024에는 HBM과 DDR5를 중심으로 더욱 우호적인 계약들이 등장할 것으로 예상됨.
