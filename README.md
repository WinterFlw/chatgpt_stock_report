신한투자증권은 2023년 3분기에 대해 롯데정보통신의 매출액과 영업이익이 모두 전년대비 성장할 것으로 전망하고 있으며, 이는 대외 사업 확대와 그룹사 프로젝트 수주 증가, 데이터 센터 실적 성장, 그리고 EVSIS 매출 증가 등 여러 요인으로 인한 것으로 분석하고 있다. 해당 증권사는 롯데정보통신의 목표주가를 37,000원으로 설정하고 있으며, 매수를 추천하고 있다.
