# 2023.03.21 stock report
## 대명에너지
DS투자증권은 재생에너지 전문 발전사업자인 대명에너지를 매수 추천하며 목표가는 32,000원이다. 대명에너지는 국내에서 시장점유율 1위를 차지하고 있으며, 계측기 설치 단지개발부터 운영까지의 경험이 있다. 발전소가 늘어나면서 발전 매출액이 증가하고 있다.
## LG생활건강
