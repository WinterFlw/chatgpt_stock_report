- 지분법이익 증가를 고려해 지배이익 전망치 상향 조정, 하지만 본업의 회복 시그널 나타나기 전까지 주가 흐름 부진할 것. 투자의견 매수와 목표주가 31.5만원을 유지함.
## 기아
- 기아의 1분기 영업이익률이 이례적으로 12%를 기록하며 호실적을 보임
- 인센티브 상승 속도는 예상보다 느리지만 볼륨 증가로 인한 이익 개선 효과는 2분기에도 지속될 것으로 예상됨
- 미국 내 EV 리스 판매 비중을 30%까지 확대할 예정
- 주가는 아직 실적 모멘텀을 충분히 반영하지 못하고 있으며, 경기 리스크 요인 감안하더라도 저평가
- 투자의견은 매수, 목표주가는 10만원 유지
