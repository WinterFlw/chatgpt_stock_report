## 녹십자
- 2023년 2분기 임시 연결실적 매출액은 4,329억원으로 전년 대비 2.3% 증가, 전분기 대비 24% 증가하였으며 영업이익은 전년 대비 80.9% 증가하였다.
- 매출 성장률이 낮아지고 있어 IVIG(면역글로불린 정맥 주사) 미국 시장 진출, 해외사업 확장, R&D 투자 등을 진행 중인데 IVIG 미국 시장 진출 등이 성장 동력 확보에 중요하다.
- IVIG 시장은 고성장하고 있으며 2024년 1월 13일 FDA 허가 결과 확인 예정이며 허가 이후 2024년 하반기 시장 출시 예정이다.
- 또한 인도네시아 플랜트 건설 사업권 승인, 브라질 수출 계약 등을 통해 해외사업 확장을 추진 중이다.
- 안정적인 비즈니스를 영위하지만 분기별 실적 변동성이 높으며 인건비, 연구개발비 등의 비용 집행으로 4분기 적자 전환 예상이다.
- R&D 파이프라인은 초기 단계이며 효율적, 집중적 R&D 투자가 필요하다.
