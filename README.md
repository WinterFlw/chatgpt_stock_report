- 2Q23 매출액은 8,063 억원으로 YoY +20.5% 증가했으나, 영업이익은 30 억원(OPM 0.4%, YoY -85.9%)를 기록했다.
- SMP 가격 하락과 건설 원자재 상승 영향으로 영업이익이 축소되었다.
- 발전용 유연탄 가격 하락은 비용을 줄였지만, 영업이익 개선에는 도움이 되지 않았다.
- 사업부별로는 발전/에너지가 가동률 79%를 기록하며 예방정비 일정을 수행했고, 건설/부동산은 원자재 상승과 인플레이션 영향으로 적자전환을 했다.
- 유리 사업은 판가 인상을 통해 OPM 개선을 했다.
