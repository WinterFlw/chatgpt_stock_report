- 그룹 내 시너지 효과를 기대하고 있으며, 현대백화점 주요 매장 내 입점 및 현대L&C와 공동 개발한 리바트윈도우(창호) 영업활동 강화, 지누스와의 중장기적 협업 등을 통해 트래픽 확대를 기대하고 있다.
## 유한양행
- CHRYSALIS 임상에서 긍정적인 결과가 예상되어 투자의견은 BUY와 목표주가는 80,000원으로 유지됨.
- CHRYSALIS 임상의 중간 데이타가 2023 ASCO 초록에서 공개될 예정이며, 장기 추적관찰 데이타에서는 타그리소보다 50%가량 Progress Free(종양진행 없음)을 기록했음.
- FLAURA2 임상 3상의 결과 발표 예정이며, CHRYSALIS 20-코호트 mPFS는 36개월 이상 예상됨.
- 2023년 하반기 발표된 MARIPOSA 임상의 중간 데이타도 긍정적인 결과가 나올 것으로 예상됨.
- 2024년 초 FDA 신약 승인 신청, 하반기 승인 예상, 2024년 말 출시로 마일스톤 유입 기대됨.
