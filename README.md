## POSCO홀딩스

## 대원전선
- 대원전선은 고부가 자동차용 전선으로 선택과 집중하며 수익성을 제고하고 있는 중견 전선업체이다.
- 전방 와이어링 하네스 시장 성장에 따른 수혜 전망이 있으며, 2023년 매출액은 5,201억원으로 예상되며, 전년대비 -7.5%의 감소가 예상되지만, 영업이익은 119억원으로 전년대비 +783.3% 예상된다.
- 투자의견과 목표가는 없다.
