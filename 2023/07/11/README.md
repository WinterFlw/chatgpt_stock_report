- 씨에스윈드는 전세계 MS 15% 이상, 미국 MS 30~40%의 타워 1위 업체이며 인력 운영에 강점을 가지고 있어 품질 관리에 탁월함
- 하부구조물 사업을 권유했을 가능성도 있어 블라트 인수 배경에는 씨에스윈드의 사업다각화 시도가 있을 수 있음
