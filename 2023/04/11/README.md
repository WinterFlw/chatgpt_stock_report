- 투자의견 매수, 목표주가 56,000원
- 가장 명확한 주주환원 정책, 높은 수익성과 자본비율, 비용 통제력, 환율 안정화 등의 이유로 탑픽으로 제시
## 신한지주
- 실적은 좋지 않지만 주가가 매우 저평가된 상태
- 매수 추천, 목표주가 44,000원
- 주주환원정책이 좋고, 비은행부문과 글로벌 부문에서 높은 이익 기여도 있음.
