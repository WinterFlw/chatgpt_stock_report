# 2023.05.16 stock report
## 삼양식품
- 2023년 1분기 매출액은 전년 대비 21.5% 증가한 2,456억원을 기록하였다.
- 내수와 수출이 둘 다 성장세를 이어가며 매출액이 두 자릿수로 증가했다.
- 하지만 외형 성장으로 인한 영업비용 증가로 인해 이익이 조금 감소했다.
- 1Q23 영업비용은 전년 대비 24.8% 큰 폭으로 증가했다.
- 이는 가동률 48.8%인 국내 밀양 공장 감가상각비 및 해외 판매법인의 온기 영업에 따른 인건비, 물류비 증가 등으로 인한 것이다.
## 인탑스
인탑스의 1분기 실적이 매출액은 1,812억원(1.4% 증가), 영업이익은 126억원(2.1% 감소), 순이익(지배주주)은 125억원으로 예상치에 거의 부합한 결과를 기록하였다. 단기적인 관점에서는 QoQ 비교가 바람직하며, 이번 실적에서는 진단키트 등의 일회성 요인이 있었다. 이베스트증권은 인탑스에 대해 Buy 추천과 40,000원의 목표가를 제시하였다.
## 이크레더블
