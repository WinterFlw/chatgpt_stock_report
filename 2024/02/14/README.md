- 매출액은 전년 대비 4% 감소하였으며 영업이익은 31% 감소하였음.
- 치과 디텍터가 2분기 연속 최대 실적을 경신했음.
- 미래에셋증권은 뷰웍스에 대해 매수 의견을 제시함.
